# Finance Suite 金融分析套件 — 使用说明

## 这是什么

Finance Suite 是一套基于 AI + 实时搜索引擎的智能分析工具集，包含 5 个专业技能模块。
它能在几分钟内完成过去需要数小时的研究和报告工作，且所有数据均来自实时搜索，有据可查。

---

## 五大技能一览

### 1. 看票分析
给出一只股票的名称或代码，自动搜索最新财报、资金流向和新闻，生成一份带数据来源的深度分析报告。偏交易视角，短平快。

**适用场景**：想了解一只股票值不值得关注时
**输入示例**：`帮我看看贵州茅台`、`分析一下比亚迪`

### 2. 宏观内参
用通俗易懂的语言解读当前宏观经济形势，自动获取最新的GDP、CPI、货币政策等数据。

**适用场景**：想了解经济大环境对投资的影响时
**输入示例**：`最近经济形势怎么样`、`央行最新政策对我们有什么影响`

### 3. 麦肯锡报告
基于你提供的会议纪要、调研数据或想法草稿，生成一份结构严谨的咨询级战略分析报告。

**适用场景**：需要做重大决策、写商业计划或汇报材料时
**输入示例**：`帮我分析这份会议纪要，出一份咨询报告`

### 4. 视频拆解
给出视频的逐字稿或链接，自动拆解为结构化的知识点、思维导图和行动清单。

**适用场景**：看完一个干货视频，想快速提炼和复习核心内容时
**输入示例**：`帮我拆解这个视频`（附逐字稿或链接）

### 5. 深度研究（行业分析 + 公司调研）
一个入口，三种模式。说一个行业名就出行业报告，说一家公司名就出企业深度调研（自动附带行业速览），两个都说就出联合报告。

**行业模式**：6维度分析框架 - 行业概览、市场规模、供需分析、竞争格局、技术路线、趋势与风险。
**公司模式**：自动搜索4份季度电话会议、年报、公司网站，生成10节完整研究报告 - 业务理解、板块拆解、产品详情、客户分析、竞争格局、行业环境、增长触发因素、关键风险、管理层言行一致性、情景推演。
**联合模式**：行业深度 + 企业深度 + 公司-行业匹配度分析。

**适用场景**：想深入了解一个行业或一家公司的全貌，达到"餐桌上能讲清楚"的程度时
**输入示例**：
- 行业：`帮我分析一下新能源汽车行业`
- 公司：`帮我研究宁德时代`、`调研一下台积电`
- 联合：`研究比亚迪在新能源汽车行业的位置`

> 和"看票分析"的区别：看票 = 快速交易视角，深度研究 = 长篇认知构建。说"看看茅台"走看票，说"研究茅台"走深度研究。

## 核心特点

- **实时数据**：接入专业搜索引擎，数据来自当天的互联网，不是过时的训练数据
- **有据可查**：每一条数据都附带来源链接，可以点击核实
- **诚实可靠**：搜索不到的数据会明确标注，绝不编造数字
- **人话输出**：专业但不晦涩，你爸妈也能看懂

## 快速上手

1. 打开 Claude Code
2. 说出你的需求（如：`看看贵州茅台`、`分析一下AI行业`）
3. 等待搜索和分析完成
4. 获得带数据来源的完整报告

无需任何技术背景，直接对话即可使用。
